# VRChat の Canny のための Cookbook

[English](COOKBOOK.md)

VRChat のフィードバックサイト (feedback.vrchat.com、Canny 上に構築) を頻繁に使っていると、目的の投稿を見つけるのに思ったより手間取る場面に出会うことがあります。この cookbook はそのような場面のための実践的なパターンを集めたものです。

以下のパターンは次のどちらか、または両方を使います:
- ブラウザ拡張 (Canny 全般のサイトで動作)
- VRChat 検索 Web ページ

両方とも https://github.com/naqtn/vrchat-feedback-boards-helper で紹介しています。

## 他の board の結果が混ざらないようにする

ある board でトピックを検索すると、他の board の結果も出てきてしまいます (board を横断するテキスト検索は Canny の機能で、VRChat は自分のサイトでこれを有効にしています)。たとえば翻訳関連の話題を探そうと Localization board で「avatar」を検索すると、他の board にある avatar の不具合や avatar 作成に関する投稿も結果に並んでしまいます。

**レシピ:** ブラウザ拡張で「Hide posts from other boards」を有効にしてください。

結果一覧は今いる board の投稿だけになります。関係のない avatar 不具合の投稿は消えます。

## 自分が過去に書いた投稿を探す

ある話題で投稿した覚えはあるのに、見つけられないことがあります。Canny の「My Own」フィルターは、自分が書いた投稿と自分が投票しただけの投稿を一緒に表示します。一覧には投稿者名が表示されないため、どれが自分のものなのかが見分けられません。

**レシピ:** ブラウザ拡張のフィルター設定を「Only posts that I wrote」に変更してください。

結果一覧には自分が書いた投稿だけが残ります。それでも数が多いかもしれませんが、自分の書いた文章を読み返すのは、見覚えのないタイトルや本文の冒頭を一つひとつ確認するよりずっと速いはずです。

## 古い投稿を探す

以前見た覚えのある投稿を探しているのに、一覧の奥深くに埋もれて見つけられないことがあります。Canny のデフォルトのソートは話題になっているものや新しいものを先に表示する仕様で、「Oldest」のソート選択肢は用意されていません。

**レシピ:** Web ページで Sort を Oldest に設定してください。

結果一覧は最も古い投稿から並びます。後述の「日付で範囲を絞り込む」の日付表示と組み合わせれば、覚えている時期にすばやく辿り着けます。

## 信頼できる投稿者の投稿を目立たせる

board の新着投稿を眺めていると、しっかり読むべき投稿がどれなのか一目では判断しにくいことがあります。有益な投稿が、そうではない投稿の中に紛れてしまいます。

**レシピ:** ブラウザ拡張で、信頼できると思う投稿者の名前を「Author Names」に追加し、「Mark posts by specific authors」を有効にしてください。その人たちの投稿に色がついて一覧で目立つようになります。

より強く絞り込みたい場合は「Only posts by specified authors」を代わりに有効にしてください。「Author Names」のリストは別の場面でも役立ちます。ある投稿を書いた人は思い出せるものの投稿自体が見つけられないとき、その人の名前をここに追加すれば一覧を絞り込めます。

## ある話題を継続的にウォッチする

数週間から数か月にわたって状況を見続けたい話題があるとします。たとえば、ある機能についてどのような報告が出ているかを継続的に把握したいような場合です。毎回同じ検索条件を入力し直すのは面倒です。

**レシピ:** Web ページで検索条件に短いメモを付けて保存してください。

保存した条件はいつでも呼び出して同じ検索を実行できます。ブラウザのブックマークでも代用できますが、保存した条件の一覧にはメモが表示されるため、複数を持つときは管理しやすくなります。

## 日付で範囲を絞り込む

投稿された時期はだいたい覚えているのに、活用できないことがあります。Canny のテキスト検索の結果は日付順に並ばず、一覧にも日付は表示されません。

**レシピ:** Web ページで Sort を Newest か Oldest に設定してください。ブラウザ拡張で「Show the date each post is created」を有効にしてください。これで一覧の各エントリの横に日付が表示されます。

覚えている時期にスクロールで移動し、それ以外の範囲の投稿は飛ばせます。(Canny の UI ではテキスト検索とソートを同時に使えませんが、VRChat 検索 Web ページではこの組み合わせが可能です。)

## VRChat が対応しないと決めた投稿を探す

VRChat が「対応しない」(won't fix) と判断した投稿、あるいはそれ以外の理由で Closed になった投稿を後から見たいことがあります。Canny のフィルター UI には Closed を選ぶ選択肢がありません。

**レシピ:** Web ページで Status を Closed に設定して検索してください。

新しい要望を投稿する前に同じ趣旨のものがすでに却下されていないかを確認するなど、背景情報を調べる用途に使えます。

## まだ評価されていない投稿を探す

まだステータスが更新されていない投稿を探したいことがあります。Canny のフィルター UI には Open を選ぶ選択肢もやはりありません。

**レシピ:** Web ページで Status を Open に設定して検索してください。

Web ページは複数のステータスを同時に指定できるので、より広く把握したい場合は Open を Tracked や Interested など処理途中のステータスと組み合わせて検索することもできます。

## すべてを組み合わせる: 古くて未解決の投稿を探す

まだ解決されていない話題の投稿を思い出したものの、見つけられないことがあります。ここではいくつかの難しさが重なっています。投稿が古くて一覧の奥に埋もれていること、ステータスが決着のついていない複数の状態 (Open, Tracked, Interested, In Progress, Needs More Information) のいずれかでありうること、テキスト検索の結果に他の board の投稿が混ざることなどです。

**レシピ:**

Web ページで:
- Sort を Oldest に設定する
- 上で挙げたステータス候補にチェックを入れる (Web ページはステータスの複数選択に対応している)
- 検索を実行する

結果はブラウザ拡張を有効にした状態のブラウザで開いてください:
- 「Hide posts from other boards」を有効にして関係のない投稿を除外する
- 「Show the date each post is created」を有効にして、明らかに新しすぎる投稿を一覧の段階で飛ばせるようにする
- 著者の心当たりがあるなら、「Author Names」にその名前を登録して「Mark posts by specific authors」を有効にする
- 自分が書いたものかもしれないなら、「Mark my posts」を有効にする

ツールを組み合わせれば、見つけにくい古い投稿も扱える範囲に収まります。

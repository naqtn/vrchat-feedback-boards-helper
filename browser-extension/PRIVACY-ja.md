# プライバシーポリシー

**Canny extension for End-users**

発効日: 2026-05-06
管理者: naqtn (<https://github.com/naqtn>)

## 要約

この拡張は、個人データを収集・送信・共有・販売しません。

## 拡張の動作

この拡張は、Canny を使ったフィードバックサイト (任意の `*.canny.io` サイトと、その別名として `feedback.vrchat.com`) を訪問しているときに限り、あなたのブラウザ内で動作します。サイトの閲覧と操作を改善するもので、たとえば各投稿の追加情報を一覧に表示したり、選んだ条件に合う投稿に印を付けたり、絞り込み条件に合わない投稿を任意で非表示にしたりします。

## デバイス内に保存されるデータ

この拡張は、オプション画面とポップアップを通じてあなたが行った設定値や構成値を、ブラウザのローカルストレージ API (`chrome.storage.local`) を使ってデバイス内に保存します。例として、投稿一覧に追加する表示項目、印を付ける投稿、絞り込みフィルタの状態、印付けや絞り込みのために登録したユーザー名のリストなどが含まれます。

このデータは:

- あなたのデバイス内に留まります。
- この拡張の開発者にも、第三者にも送信されません。
- 拡張をアンインストールすることで、またはブラウザの設定から拡張のストレージを削除することで、いつでも消去できます。

この拡張はアカウント認証情報を保存しません。上述の構成値の他には、永続的なデータは保存しません。

## ネットワーク通信

表示機能をサポートするため、この拡張は、あなたが現在閲覧している Canny サイトから追加のページを取得する場合があります (たとえば、投稿一覧に表示されない各投稿の情報を読み取るため)。これらの通信は:

- あなたのブラウザから、Canny サイトとの既存のセッションを使って発行されます。
- この拡張の開発者にも、第三者にも送信されません。
- あなた自身がそれらのページを開くのと、機能的に同等です。

この拡張は、開発者のサーバーを含む他のいかなるサーバーにも接続しません。解析・テレメトリ・クラッシュレポート・広告のコードはいずれも含みません。

## データの共有・販売

この拡張は、誰とも・誰にもデータを共有したり販売したりしません。共有するためのデータを収集していないためです。

## 将来のバージョン

このポリシーは拡張の挙動を一般的な言葉で記述しており、各セクションの具体例は発効日時点で利用可能な機能を反映したものです。将来のバージョンで機能の追加や名称変更が行われる場合がありますが、ここに述べられている原則 (個人データを収集しない、開発者や第三者へ送信しない、設定値はあなたのデバイス内のみに保存する) は、ポリシー自体が更新されない限り、それらの変更にも適用されます。
